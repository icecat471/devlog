Hugo syntax highlight 변경

April 16, 2021 · 1 min · icecat471
Table of Contents

  • 2. css 찾기

1. 원하는 테마 찾기#

highlightjs.org에서 원하는 테마를 찾는다.

2. css 찾기#

highlightjs github에서 원하는 테마의 css파일 내용을 복사한다.

3. 테마 변경#

config.yml에 아래의 내용을 추가한다.

pygmentsUseClasses: true

(Hugo PaperMod 테마 기준) /themes/PaperMod/assests/css/hljs/an-old-hope.min.css 의 내용을 복사한 내용으로 바꾼다.

=> 임시적으로 설정한것으로 theme overriding으로 변경 가능한지 찾아봐야 함.
